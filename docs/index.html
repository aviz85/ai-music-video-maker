Capabilities

Built for creators

Everything you need to turn audio into professional music videos.

Word-Level Timing

ElevenLabs provides precise word-by-word timestamps for perfect lyrics synchronization. Used as ground truth for all timing alignment.

Gemini Audio Analysis

AI-powered deep listening identifies vocals, instrument highlights, musical structure, and optimal video segments.

9 Camera Angles

Single 4K collage generates 9 unique camera angles: wide stage, singer closeup, instruments, crowd, and more.

LTX Video Generation

State-of-the-art AI video generation using fal.ai LTX-2 for realistic, audio-synced video clips.

Lyrics Overlay

Animated lyrics with multiple styles: karaoke highlighting, fade-in effects, and minimal mode. Full RTL language support.

Smart Shot Selection

Audio-driven shot selection: vocals show the singer, guitar solo shows the guitarist. What you hear is what you see.

Setup

Installation

Install the skill for Claude Code in a few simple steps.

Terminal

# Clone the repository
git clone https://github.com/aviz85/ai-music-video-maker.git
cd ai-music-video-maker

# Install dependencies
cd .claude/skills/audio-to-video/scripts
npm install

# Return to project root
cd ../../../..

# Create .env with your API keys
cat > .claude/skills/audio-to-video/scripts/.env << EOF
FAL_KEY=your_fal_ai_key_here
GEMINI_API_KEY=your_gemini_key_here
EOF
