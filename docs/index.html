Installation

Install the AI Music Video Maker skill for Claude Code in just a few steps.

Terminal

# Clone the repository
git clone https://github.com/aviz85/ai-music-video-maker.git
cd ai-music-video-maker

# Install dependencies for audio-to-video scripts
cd .claude/skills/audio-to-video/scripts
npm install

# Go back to project root
cd ../../../..

# Create .env file with your API keys
cat > .claude/skills/audio-to-video/scripts/.env << EOF
FAL_KEY=your_fal_ai_key_here
GEMINI_API_KEY=your_gemini_key_here
EOF

# Optional: For lyrics transcription (ElevenLabs)
# Create .env in transcribe skill if you have it

Required API Keys

GEMINI_API_KEY - For audio analysis and storyboard generation
Get it from Google AI Studio
FAL_KEY - For LTX video generation
Get it from fal.ai
ELEVENLABS_API_KEY - For word-level transcription (optional, for lyrics)
Get it from ElevenLabs

Quick Start

Start creating AI music videos in minutes with Claude Code.

Usage with Claude Code

1

Open the project in Claude Code

Navigate to the ai-music-video-maker directory and start Claude Code.

cd ai-music-video-maker && claude

Ask Claude to create a music video

Claude will use the music-video skill to orchestrate the entire pipeline.

Create a music video from /path/to/song.mp3

Customize as needed

Specify duration, style, or use "cheap mode" for faster generation.

Add lyrics overlay (optional)
