// Capabilities

Everything in one pipeline

From raw audio to professional music video. No manual editing required.

01

Word-Level Timing

ElevenLabs provides precise word-by-word timestamps. Ground truth for all alignment.

02

Gemini Audio Analysis

AI identifies vocals, instruments, musical structure, and optimal segments.

03

9 Camera Angles

Single 4K collage generates 9 unique angles: stage, closeups, crowd, silhouettes.

04

LTX Video Generation

State-of-the-art AI video via fal.ai LTX-2. Audio-synced, realistic motion.

05

Lyrics Overlay

Animated lyrics: karaoke, fade-in, minimal modes. Full RTL language support.

06

Smart Shot Selection

Audio-driven cuts. Vocals show singer. Guitar solo shows guitarist.

// Setup

Installation

Clone, install, configure. Ready in 2 minutes.

terminal 
# Clone the repository
git clone https://github.com/aviz85/ai-music-video-maker.git
cd ai-music-video-maker

# Install dependencies
cd .claude/skills/audio-to-video/scripts
npm install

# Return to project root
cd ../../../..

# Configure API keys
cat > .claude/skills/audio-to-video/scripts/.env << EOF
FAL_KEY=your_fal_ai_key
GEMINI_API_KEY=your_gemini_key
EOF
